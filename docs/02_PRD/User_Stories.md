# User Stories — StyleMind

## 1. Guest Stories

```text
As a guest,
I want to browse fashion products,
so that I can explore available items before signing up.
```

```text
As a guest,
I want to add products to cart,
so that I can prepare my purchase before login.
```

## 2. Customer Stories

```text
As a customer,
I want to register and login,
so that I can buy products and track my orders.
```

```text
As a customer,
I want to manage delivery addresses,
so that I can checkout faster.
```

```text
As a customer,
I want to chat with an AI stylist,
so that I can receive outfit recommendations.
```

```text
As a customer,
I want to track my orders,
so that I know my purchase status.
```

## 3. Admin Stories

```text
As an admin,
I want to create and update products,
so that the catalog is always up to date.
```

```text
As an admin,
I want to manage categories,
so that products are organized clearly.
```

```text
As an admin,
I want to manage AI index jobs,
so that AI recommendations can use fresh catalog data.
```

```text
As an admin,
I want to view notification logs,
so that I can monitor business operations.
```
